# ELS 완전 정복 — 교육 웹사이트

고등학생을 위한 ELS(주가연계증권) 설명 웹사이트입니다.

## 서버 실행 방법

### 방법 1 — Python (추천)

```bash
cd frontend
python3 -m http.server 8080
```

브라우저에서 http://localhost:8080 접속

### 방법 2 — Node.js (npx)

```bash
cd frontend
npx serve .
```

### 방법 3 — VS Code Live Server

VS Code에서 `index.html` 파일을 열고 우측 하단 **Go Live** 버튼 클릭
